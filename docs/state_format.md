## Board JSON 10x9 format

- Board là mảng 10 hàng × 9 cột
- Mỗi ô là string hoặc rỗng ""

### Piece encoding
- r = RED
- b = BLACK

| Piece             | Code |
|------             |------|
| King (Tướng)      | K |
| Advisor (Sĩ)      | A |
| Elephant (Tượng)  | E |
| Rook (Xe)         | R |
| Knight (Mã)       | N |
| Cannon (Pháo)     | C |
| Pawn (Tốt)        | P |

### Example
"rR" = Red Rook  
"bK" = Black King
